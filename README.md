# NS Platform

Платформа определения целеполагания

## Описание
NS Platform - это веб-приложение для управления целями и сессиями целеполагания.

## Технологии
- Python 3.12
- Django 6.0
- PostgreSQL
- Poetry (управление зависимостями)
